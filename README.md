Step Style

Projeto de um site institucional com proposta de e-commerce no segmento de moda streetwear, desenvolvido utilizando HTML, CSS e JavaScript.

O objetivo do projeto é apresentar uma estrutura moderna, responsiva e organizada de loja virtual, aplicando conceitos fundamentais de desenvolvimento front-end.

Sobre o Projeto

O Step Style simula uma loja online voltada para roupas, calçados e acessórios. A aplicação contém:

Página inicial com seção hero (banner principal)

Seção de categorias de produtos

Área de produtos em destaque com renderização dinâmica

Seção de benefícios comerciais

Formulário de newsletter

Página de login

Estrutura modular de arquivos

O projeto foi desenvolvido com foco em organização de código, boas práticas de estrutura HTML e estilização responsiva.

Tecnologias Utilizadas

HTML5 – Estrutura semântica da aplicação

CSS3 – Estilização, layout com Flexbox e Grid

JavaScript – Interatividade e manipulação do DOM

Google Fonts (Inter) – Tipografia
